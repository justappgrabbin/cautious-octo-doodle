# West-First Source-Preserving Audit

## What changed
I switched from organ extraction-first to source-preservation-first. The package keeps whole donors intact, then maps unique patterns and their intended placement.

## Why
Because the substrate topology matters. Pulling pieces apart before knowing their role risks losing message-passing, MCP hub behavior, registries, and living continuity.

## Critical correction
MCP is represented as a message-passing channel inside the hub, not as a detached tool registry. App Center is the visible registry face.

## Placement map
- `message-hub.js` — EventBus, Message Possum, MCP envelope routing, AI/provider message passing. MCP lives here as protocol/channel, not detached tool list.
- `app-center-registry.js` — Visible registry face: apps, files, agents, MCP tools, ingested organs, installed app list, source inventory launchers.
- `source-file-registry.js` — Canonical source inventory and donor ancestry. Keeps every source file addressable and restorable.
- `agent-registry.js` — Agent identity/profile/skills registry and AI participant records.
- `mcp-morph-bridge.js` — MCP discovery/call/execute translated into morph message envelopes and routed through message hub.
- `app-editor-construct.js` — GrapesJS/WebStudio/Plasmic/App Builder construct tools.
- `svg-studio.js` — SVG maker and vector/shape editing organ.
- `mesh-matrix.js` — Persistent 5-layer/5D mesh, node relation field, MRNN/crossmodal matrix.
- `substrate-bridge.js` — Bridge to sealed Synthia substrate. Synthia source is preserved untouched and mounted, not picked apart.
- `ingestion-runtime.js` — File/text/zip ingestion, atomization, ancestry registration, donor digestion pipeline.
- `media-tools.js` — Photo/image/media tools and canvas viewers/editors.
- `terminal-files-workspace.js` — Terminal, file explorer, workspace panels.
- `external-ai-bridges.js` — HuggingFace, Expo/mobile, backend/trident URLs, voice/audio, external AI connectors.
- `morphic-overlay.js` — PhonyOS/morph lens, device modes, reality strip, atmosphere color/pulse.
- `keyboard-ime-sidecar` — MorphKeyboard Android IME sidecar source preserved separately.
